🎆 Diwali-Sales-Data-Analysis:

📌 Project Overview:
This project performs Exploratory Data Analysis (EDA) on a Diwali sales dataset using Python.
The goal is to understand customer purchasing behavior during the Diwali festival and discover useful business insights.

🎯 Objectives:
Analyze customer demographics (age, gender, marital status)
Identify top spending groups
Find high-performing product categories
Understand regional/state-wise sales trends
Generate insights for better marketing decisions

🛠️ Tools & Libraries Used:
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook

📂 Dataset Information
The dataset contains:
Customer details (Age, Gender, Marital Status, Occupation)
Location data (State, Zone)
Product categories
Order counts
Purchase amounts

🔍 Data Cleaning Steps
Removed empty columns
Handled missing values
Converted data types
Verified dataset structure

📊 Analysis Performed
Gender-wise spending analysis
Age group vs spending
Marital status comparison
State-wise sales analysis
Product category performance

💡 Key Insights
Females spent more than males during Diwali shopping
Age group 26–35 had the highest spending
Married customers showed higher purchase amounts
Certain states contributed significantly to total sales
Food, Clothing, and Electronics were top categories

📈 Sample Visualizations
The project includes:
Bar charts
Count plots
Sales distribution graphs
